## [v2 重构实战：跨语言全栈迁移](/cases/v2-refactor)

Java 老手、Python 新手，用 AI 协作在 1 天内完成 Java→Python / Vue 2→Vue 3 的全栈重构。14 条编码规则让 AI 从"不可控"变成"可控"。

- **规模**：14,983 行 / 143 文件 / 56 个 commit
- **人工投入**：~88 分钟
- **亮点**：CRUD 27 表 1 天完成、前端 14 页面 54 分钟、规则飞轮效应
