# 示例文档

这里放的是虚构/合成示例，用来展示 `standard spec`、`light spec` 和 `codemap` 在实际任务中的写法。

- [`specs/spec-standard-security-status-race.md`](specs/spec-standard-security-status-race.md)
  - 标准版 spec 示例，展示如何沉淀目标、边界、事实、计划、验证与回写闭环。
- [`specs/spec-light-runtime-compat.md`](specs/spec-light-runtime-compat.md)
  - 轻量版 spec 示例，展示强模型场景下如何用最小信息维持任务理解、核心目标与进度同步。
- [`codemap/codemap-feature-content-control.md`](codemap/codemap-feature-content-control.md)
  - 功能级 codemap 示例，展示如何快速定位入口、路由、关键链路、数据结构与风险点。

说明：

- 这些文档是为教学目的构造的合成案例。
- 编写原则是保留问题形态、推理过程、修复方式与验证思路，但不映射任何具体项目、仓库、日志样本或测试标识。
